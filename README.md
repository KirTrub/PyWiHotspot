# PyHotspot (Vibecoded)

GUI приложение для создания Hotspot.

## Особенности

* **One-click Start**: Автоматическое создание и запуск Hotspot.
* **Traffic Monitor**: Графики скорости загрузки и отдачи в реальном времени.
* **Device List**: Список подключенных устройств с именами и MAC-адресами.
* **QR Code**: Мгновенное подключение мобильных устройств.
* **Autostart**: Создание системного сервиса для запуска при загрузке ОС.

## Установка

### Arch Linux (AUR)

Пакет доступен в AUR.

Установка через `yay`:

```bash
yay -S pyhotspot
```

Установка через `paru`:

```bash
paru -S pyhotspot
```

### Установка из исходников

Клонируйте репозиторий:

```bash
git clone https://github.com/youruser/PyWiHotspot.git
cd PyWiHotspot
chmod +x ./install.sh
./install.sh
```

После установки приложение появится в общем списке приложений.
